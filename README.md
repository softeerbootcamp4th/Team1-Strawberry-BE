# 그라운드 룰

1. 퇴근 전 회고
2. PR 시 코드 리뷰
3. 칭찬 무조건 2가지 이상 ( 코드 외 가능)
4. 아쉬운 점 무조건 2가지 이상 (코드 외 가능)
5. 월요일마다 주간 리뷰
6. 막히는 부분 있으면 페어프로그래밍 하기
7. 이슈 (궁금한 점, 모르는 것 등등) 발생 시 위키 기록 후 정리 및 공유
